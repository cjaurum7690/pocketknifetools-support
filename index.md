# PocketKnife Tools

PocketKnife Tools is a lightweight utility toolkit designed to make everyday tasks faster and easier.

## Features
- Weather Tracker by City
- Unit Converter
- Currency Converter
- Language Translator
- Compass
- Flashlight with Morse Code Functionality
- Leveler
- Dictionary
- List Maker
